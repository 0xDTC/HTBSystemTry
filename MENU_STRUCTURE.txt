INTERACTIONS:

1. Click tray icon → Main menu opens
2. Hover over item with ▶ → Submenu expands
3. Click item → Action executes
4. Click outside menu → Menu closes

NOTIFICATIONS:

All actions trigger desktop notifications:
  • Spawning machine: "Manager is spawning! Ready in ~2-3 min"
  • VPN download: "Downloaded US VIP 1 to ~/Downloads/htb-vpn/"
  • Flag submit: "🎉 Correct Root flag! +40 points"
  • Errors: "❌ Failed to spawn: Machine already active"

FLAG SUBMISSION FLOW:

1. Click: 🚩 Submit Flag
2. Dialog appears (zenity): "Enter flag for Keeper:"
3. Type flag: HTB{...}
4. Another dialog: "Select flag type:"
   - User Flag
   - Root Flag
5. Click selection
6. Notification: "✓ Correct User flag!" or "✗ Incorrect"

═══════════════════════════════════════════════════════════════════

TYPICAL WORKFLOW:

Morning Routine:
  1. Login to Kali
  2. HTB Tool auto-starts (tray icon appears)
  3. Click icon
  4. VPN → US Free 1 → Download TCP
  5. Terminal: sudo openvpn ~/Downloads/htb-vpn/htb_vpn_123_tcp.ovpn
  6. Machines → Active → Keeper → Spawn
  7. Wait notification: "Keeper is spawned!"
  8. Start hacking!

After Pwn:
  1. Click tray icon
  2. Machines → Active → Keeper
  3. Submit Flag
  4. Enter: HTB{user_flag_here}
  5. Select: User Flag
  6. Notification: "🎉 Correct!"
  7. Continue to root...
  8. Submit Flag → Enter root flag
  9. Select: Root Flag
  10. Notification: "🎉 Correct!"
  11. Keeper → Stop
